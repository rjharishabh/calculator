How To Calculate   Hide

Let the set of n terms be x1, x2, x3, x4, x5, x, ......, xn.

Let the Maximum Value be x9 and the Minimum Value be x4.

Then the Range will be x9 - x4.

$$Sum = \sum_{i=1}^n x_n = x_1 + x_2 + x_3 + ... + x_n$$ $$Mean,\hspace{1cm}\mu = \frac{1}{n}\sum_{i=1}^n x_n = \frac{1}{n}(x_1 + x_2 + x_3 + ... + x_n)$$ In other words, $$Mean,\hspace{1cm}\mu = \frac{Sum \hspace{0.2cm}of \hspace{0.2cm}terms}{Number\hspace{0.2cm} of\hspace{0.2cm} terms}$$

For finding the Median, first sort the array in the Ascending or Descending Order.Then

if the number of terms is odd, then the median will be (\(\frac{n\hspace{0.1cm}+\hspace{0.1cm}1}{2}\)) th term.

if the number of terms is even, then the median will be \(\frac{1}{2}\)(\(\frac{n}{2}\) th term + \((\frac{n}{2}\hspace{0.1cm}+\hspace{0.1cm}1)\) th term).

The term with Maximum Occurences will be Mode.

$$Variance,\hspace{1cm}\sigma^2 = \frac{1}{n}\sum_{i=1}^n (x_i\hspace{0.1cm}-\hspace{0.1cm}\mu)^2 $$ $$Standard \hspace{0.3cm}Deviation,\hspace{1cm}\sigma = \sqrt{\sigma^2} = \sqrt{\frac{1}{n}\sum_{i=1}^n (x_i\hspace{0.1cm}-\hspace{0.1cm}\mu)^2}$$
